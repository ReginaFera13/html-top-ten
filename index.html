HTML

Here are a few HTML things I like:

img

Describe this term, and make an example of how its used:

The <img> tag is a self-closing tag that lets us add images to our site. This is a great way to easily add some visuals!

Code example:

          <img src="./name-of-file.jpg">

table

Describe this term, and make an example of how its used:

The <table></table> tag lets us add tables to our site. This is a great way to display data!

Code example:

          <table>
            <thead>
              <tr>
                <th>Name</th>
                <th>Age</th>
              </tr>
            </thead>
            <tbody>
              <tr>
                <td>Alice</td>
                <td>35</td>
              </tr>
              <tr>
                <td>Bob</td>
                <td>29</td>
              </tr>
            </tbody>
          </table>

Example of use:

With the <table></table> tag, the code above displays as:

Name Age
Alice 35
Bob 29

blockquote

Describe this term, and make an example of how its used:

The <blockquote></blockquote> tag lets us add long quotes to our site. This is a great way to display historical quotes or customer references!

Code example:

          <blockquote cite="https://en.wikiquote.org/wiki/J._R._R._Tolkien">
            My political opinions lean more and more to Anarchy (philosophically understood, meaning abolition of control not whiskered men with bombs) … the most improper job of any man, even saints (who at any rate were at least unwilling to take it on), is bossing other men. Not one in a million is fit for it, and least of all those who seek the opportunity.<br />- JRR Tolkien
          </blockquote>

Example of use:

With the <blockquote></blockquote> tag, the code above displays as:

CSS Things

Here are a few CSS things I like:

display

Describe this term, and make an example of how its used:

The display: display-type; tag lets us choose the display style of the element. This is how you would change the display to something like flex so that we can have a more dynamic layout!

Code example:

          body {
            display: flex;
          }
resource link

font-family

Describe this term, and make an example of how its used:

The font-family: font-family-type; tag lets us choose the font of the element to something like sans-serif!

Code example:

          body {
            font-family: sans-serif;
          }
resource link

text-decoration

Describe this term, and make an example of how its used:

The text-decoration: text-decoration-type; tag lets us choose the text decoration of the element. This is how you would change a hyperlink to not have the underline!

Code example:

          a {
            text-decoration: none;
          }
resource link

box-shadow

Describe this term, and make an example of how its used:

The box-shadow: box-shadow-type; tag lets us add a box-shadow to the element. This gives it a more 3D feel!

Code example:

          body {
            box-shadow: 2px 5px gray;
          }
resource link

list-style

Describe this term, and make an example of how its used:

The list-style: list-style-type; tag lets us remove the bullets in a list!

Code example:

          ul {
            list-style: none;
          }
